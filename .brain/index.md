# Brain Vault

Project knowledge base curated by Brain daemon.

## Structure

- **episodic/** - Rolling recent history (decisions, failures, successes)
- **semantic/** - Long-term project knowledge (architecture, entities, constraints)
- **procedural/** - How to do things (patterns, anti-patterns)
- **learned/** - From user corrections (preferences, corrections)
- **issues/** - Problems and workarounds (active, resolved)
- **sessions/** - Session summaries by date/topic

## Usage

Hot context appears in CLAUDE.md. This vault contains full detail.
Links in CLAUDE.md point here for progressive disclosure.
